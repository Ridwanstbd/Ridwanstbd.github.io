PASTIKAN DEVICE ANDA TERINSTALL NODE JS

Untuk menjalankan kode ini terlebih dulu:
1. Install tailwind css dengan mengetikan kode berikut di terminal:
npm init -y
npm install -D tailwindcss
npx tailwindcss init

2. Jalankan Node.js dengan ketik kode berikut diterminal:
npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
